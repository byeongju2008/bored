# 🚀 프로젝트명: Bored?

## 📌 한 줄 소개

심심할 때 같이 할 사람을 빠르게 찾는 웹사이트

---

## 🎯 문제 정의

요즘 사람들은 할 것은 많지만 같이 할 사람을 찾기 어렵다.
디스코드는 친목 위주라 처음 들어가기 어렵고,
새로운 사람을 쉽게 구하는 구조가 아니다.

---

## 💡 해결 방법

사용자가 방을 만들어
같이 할 사람을 모집하고
바로 채팅 및 음성으로 연결되는 웹사이트를 만든다.

---

## 🔥 핵심 기능

### 1. 사람 모집 시스템

* 방 생성
* 방 목록
* 태그 기반 검색

### 2. 실시간 소통

* 채팅
* 음성 채팅

### 3. 방 관리 시스템

* 방장 권한 (킥, 밴, 음소거)
* 경고 시스템 (사용자 설정 가능)
* 규칙 설정

### 4. 사용자 편의 기능

* 대기방
* 프로필 상태 표시
* 간단한 UI

---

## ⚙️ 기술 스택

* Backend: Python, Flask
* Frontend: HTML, Tailwind CSS, JavaScript
* Database: SQLite

---

## 🧩 현재 설계 구조

### 메인 페이지

* 방 목록
* 검색창
* 방 만들기 버튼

### 방 생성 페이지

* 방 제목
* 설명
* 태그
* 최대 인원
* 경고 시스템 설정
* 규칙 작성

### 방 내부

* 채팅
* 음성 채팅
* 사용자 목록
* 방 설정
* 규칙 보기
